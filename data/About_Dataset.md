# Dataset Notice

## Dataset Availability

Due to copyright restrictions, the dataset used in this project cannot be publicly shared or redistributed. The images were collected from publicly available sources, but their original licenses and ownership prevent redistribution as part of this repository.

## Data Collection Methodology

The dataset was created using a combination of:

* Automated web scraping techniques to gather candidate images
* Manual filtering and curation to ensure quality and relevance
* Selection of images representing both domains (live-action and cartoonized styles)

This process was necessary to build a suitable dataset for training and fine-tuning the GAN model while maintaining visual consistency.

## Reproducibility

To ensure transparency and reproducibility:

* The code used for data collection and preprocessing is available in the `notebooks/` directory.
* These notebooks document the scraping process, filtering steps, and dataset preparation pipeline.
* Users can recreate a similar dataset by running the provided notebooks and adapting them as needed.

## Note

Users are responsible for ensuring compliance with copyright and licensing terms when collecting and using their own data.
